# Консольная банковская система (Bank System CLI)

Проект написан на Java (Core) с использованием объектно-ориентированного программирования (ООП) и коллекций в памяти.

## 📦 Инструкция по запуску
1. Откройте проект в IntelliJ IDEA.
2. Перейдите в папку `src/main/`.
3. Запустите файл `Main.java` (нажмите зелёный треугольник Run).
4. Управляйте системой через консольное меню.
