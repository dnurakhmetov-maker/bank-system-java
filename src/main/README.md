# Консольная банковская система (Bank System CLI)

Индивидуальный проект по курсу Java (Core). Имитация работы банковской системы с управлением клиентами, счетами и транзакциями.

## 🚀 Технологический стек
* **Язык:** Java 17 / 21
* **Архитектура:** Разделение на слои (Model-Service-Main)
* **Хранение данных:** В памяти (Коллекции `ArrayList`, `HashMap`)

## 🛠️ Функционал приложения
1. **Управление клиентами:** Регистрация пользователей в системе.
2. **Управление счетами:** Автоматическая генерация номеров счетов (`ACC-1001` и т.д.), поддержка типов `DEBIT` и `CREDIT`.
3. **Финансовые операции:** Пополнение, снятие, безопасные переводы между счетами с полной валидацией баланса.
4. **Аналитический модуль:** Просмотр истории транзакций по счету, подсчет общего баланса клиента, поиск топ-клиентов.

## 📦 Инструкция по запуску
1. Склонируйте репозиторий или скачайте архив с кодом.
2. Откройте проект в любой Java IDE (например, IntelliJ IDEA).
3. Убедитесь, что в настройках проекта установлен JDK версии 17 или выше.
4. Запустите класс `src/main/Main.java`.
5. Следуйте инструкциям интерактивного меню в консоли.
